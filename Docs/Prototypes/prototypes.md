Protótipos de Interface - Petbem

Este documento apresenta a concepção visual do sistema, focada na usabilidade para voluntários em campo e administradores de abrigos.

## 1. Inspiração e UX (Experiência do Usuário)
Nossa interface foi projetada com base na análise real de perfis de proteção animal. 

**Pontos de dor identificados:**
* Necessidade de visualização rápida das fotos dos animais para evitar confusão na medicação.
* Importância de uma linha do tempo clara para que o adotante veja a evolução do comportamento.

## 2. Protótipo Mobile (App do Voluntário)
O foco da versão mobile é a entrada rápida de dados à beira do leito/baia.
* **Tela de Prontuário:** Lista de animais com status de saúde (Cores: Verde/Amarelo/Vermelho).
* **Tela de Registro:** Botões grandes para marcar "Medicação Aplicada" e campo de texto para "Comportamento do Dia".

## 3. Protótipo Web (Painel Administrativo)
Focado em gestão e relatórios de impacto social.
* **Dashboard:** Gráficos de animais recuperados e prontos para adoção (ODS 11).
* **Módulo de Exportação:** Botão para gerar o PDF completo do histórico do animal.

---
*Nota: A interface prioriza a simplicidade de uso sob luz solar (alto contraste) e facilidade de navegação com apenas uma mão.*
